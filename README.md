# DBMigrator-AI

**AI-Powered Database Migration Management Tool**

DBMigrator-AI is a comprehensive database migration assistant built with MiMo V2.5 intelligence. It provides schema diff visualization, migration status tracking, and real-time execution logging to streamline your database operations.

## Features

- **Schema Diff Viewer** — Visual comparison of source vs target database schemas with color-coded changes
- **Migration Status Dashboard** — Track pending, running, and completed migrations with progress indicators
- **Execution Log** — Real-time terminal-style log output for migration operations
- **AI-Powered Analysis** — Intelligent detection of breaking changes and migration risks

## Tech Stack

- Vanilla HTML/CSS/JavaScript
- No external dependencies
- Professional DB admin theme with dark mode

## Getting Started

Open `index.html` in any modern browser to begin managing your database migrations.
